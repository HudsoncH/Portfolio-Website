# Images go here

Drop your real photos/screenshots in this folder (subfolders are fine too),
then reference them from `data/projects.js` or your pages with a path like
`/images/your-file.png` (paths under `public/` are served from the site root,
so you do NOT include `public` in the path).

This file only exists so the empty folder isn't skipped by git — delete it
once you've added real images if you want.
